## Опис / Description

Овој проект претставува систем за структурирање, анализа и предвидување на податоци за јавни набавки во Северна Македонија со користење на Semantic Web технологии и машинско учење.  
Податоците се трансформираат во **knowledge graph** преку RDF и онтологии, овозможувајќи напредни семантички пребарувања и анализа.

Дополнително, проектот е проширен со **интерактивен веб систем**, кој овозможува интерактивен приказ, пребарување и визуелизација на податоците, како и АИ асистент.

This project presents a system for structuring, analyzing, and predicting public procurement data in North Macedonia using Semantic Web technologies and machine learning.  
The data is transformed into a **knowledge graph** using RDF and ontologies, enabling advanced semantic querying and analytics.

Additionally, the project is extended with a **interactive web system**, enabling interactive exploration and visualization of the data, as well as AI assistent.

- SPARQL

- Transformer models (multilingual embeddings)
- Pandas / NumPy

- Web technologies (Django + Bootstrap)

## Клучни компоненти / Key Components

- Data aggregation and preprocessing pipeline  
- Ontology design for public procurement  
- CSV → RDF трансформација со RML  
- SHACL валидација на податоци  
- Knowledge Graph (RDF triples)  
- SPARQL queries за анализа  
- Machine Learning модел за предвидување на вредност на договори  
- Визуелизација на трендови
- AI Assistent

---

## Анализа и функционалности / Analysis & Features

- Семантичко пребарување на договори  
- Анализа на институции и добавувачи  
- Детекција на трендови во јавни набавки  
- Идентификација на најголеми договори  
- Статистичка анализа (mean, median, std)  
- Предвидување на вредност на нови договори  
- Визуелизација на историски податоци
- AI Assistent

---

## Machine Learning модел / Machine Learning Model

- Multilingual sentence embeddings  
- Similarity search со FAISS  
- K-nearest neighbors пристап (top-K слични договори)  
- Проценка базирана на медијана  

**Резултати:**
- Подобри перформанси од baseline (median)  
- Намален RMSE и MAE  
- Подобро фаќање на шаблони во податоците

## Резултати / Results

- Успешна трансформација на табеларни податоци во knowledge graph / Successful transformation of tabular data into a knowledge graph
- Подобрена транспарентност и анализа на јавни набавки / Improved transparency and analysis of public procurement

## Заклучок / Conclusion

Проектот демонстрира како комбинацијата на **Semantic Web технологии и машинско учење** може значително да ја подобри анализа на јавни набавки.  
Со трансформација во knowledge graph и додавање на predictive analytics, системот овозможува подобра транспарентност, анализа и донесување одлуки.

The project demonstrates how combining **Semantic Web technologies and machine learning** can significantly improve public procurement analysis.  
By transforming data into a knowledge graph and integrating predictive analytics, the system enhances transparency and decision-making.
